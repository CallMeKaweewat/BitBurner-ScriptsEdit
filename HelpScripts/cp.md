
Copy a file
  
**Signature:**

  
```typescript

cp [src] [dst]

```

  

## Parameters

  
- `cp`: Copy a file
- `scr`:  name script .
- `dst`: destination.


**Example:**


  
```typescript

cp hack.js hackCopy.js

```

  
**Result:**


```Terminal
  
[HOME /]> cp hack.js hackcopy.js
File hack.js copied to hackcopy.js

```
