## expr
Evaluate a mathematical expression
  
**Signature:**

  
```typescript

expr [math expression]

```

  

## Parameters

  
- `expr`: function calculator
- `math expression`:  Input number 


**Example:**


  
```typescript

expr 42+39*45/58

```

  
**Result:**


```Terminal
  
[home /]> expr 42+39*45/58
72.25862068965517

```
