## download
Downloads scripts or text files to your computer
  
**Signature:**

  
```typescript

download [script/text file]

```

  

## Parameters

  
- `download`: dowload file
- `script/text file`:  script or text file


**Example:**


  
```typescript

download hack.js

```

  
**Result:**


```Note
  
It's download script in game to ur computer
```
