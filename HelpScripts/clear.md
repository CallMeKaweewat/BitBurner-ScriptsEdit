## clear
Clear all text on the terminal
  
**Signature:**

  
```typescript

clear

```

  

## Parameters

  
- `clear`: clear text on terminal


**Example:**

  
```Terminal

connect n00dles
run nuke.exe
nano hack.js
run hack.js -t 2

```

  
```typescript

clear
```
